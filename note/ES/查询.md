# SearchAPI



大多数search api都支持多index，多type



**routing**

当进行查询时，会将查询广播到所有的index/分片。可以通过routing参数来指定查询哪些分片。







































# QueryDSL

```console
GET /customer/external/1?pretty
```

* 排序

**QueryDSL**



query来定义query

from

size，一次查询默认返回10个documment，可以通过size指定
